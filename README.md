由于经常忘记自己的QQ密码每次都需要用手机号找回，使用第三方的软件始终感觉有点不放心，因此开发了个自用密码备份工具，软件无联网权限，无任何联网功能，可放心大胆的使用备份自己的私人密码。

功能总览
- 使用sqlcipher加密数据库
- 支持增加/修改/删除密码
- 支持多语言切换
- 支持指纹解锁/手势解锁app
- 支持主题色切换
- 使用Getx
